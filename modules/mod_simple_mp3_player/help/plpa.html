5 easy Steps

  1. Create a folder named mp3 in your Joomla! directory.
  2. Create playlist.txt (see example before) - don't forget to use UTF-8 encoding!
  3. Save your playlist to the mp3 directory.
  4. The path now should be something like that: http://www.yourdomain.com/joomla_directory/mp3/playlist.txt
  5. Finally you should copy an empty index.html to your MP3 Directory to avoid someone can see the Files of the Directory in his Browser.
