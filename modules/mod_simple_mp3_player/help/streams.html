Does it work with Streams???

Interesting question!!!

Though the Player was not explicitly developed for this Task, there is a Trick that worked well for me.

I'll show you what I've done with the m3u File of a german Radiostation (WDR2):

  1. First I've downloaded it (http://www.wdr.de/wdrlive/media/wdr2.m3u).
  2. Next I opened it in a simple Text Editor and copied the first URL (http://gffstream.ic.llnwd.net/stream/gffstream_w17b).
  3. Finally I've pasted the URL into my Playlist.

My Playlist

http://gffstream.ic.llnwd.net/stream/gffstream_w17a|My first Radio Stream - WDR2

You can do that with multiple URL's.
